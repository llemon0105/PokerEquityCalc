# PokerEquityCalc

A Texas Hold'em equity calculator built in Java. Given your hole cards and the 
community cards, it simulates thousands of random runouts using Monte Carlo 
simulation to estimate your probability of winning against an opponent's hand range.

## Features
- Full 52-card deck representation
- Hand evaluator (pairs, flushes, straights, full houses, etc.)
- Monte Carlo simulation engine (10,000+ runouts)
- Opponent range input (e.g., top 15% of hands)
- Win/loss/tie equity output

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `Main.java`

## Technologies
- Java 26
- IntelliJ IDEA
